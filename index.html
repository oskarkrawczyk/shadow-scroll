ShadowScroll

ShadowScroll will add top and/or bottom shadows to your overflowed scroll area in order to indicate that there's still something to scroll.

Setup

You can install ShadowScroll via Bower with bower install ShadowScroll or download shadow-scroll.js and include it the old-fashined way.

CoffeeScript

shadowScroll = document.querySelectorAll '.shadowScroll'
new ShadowScroll shadowScroll,
  prefix: 'shadowScroll'

JavaScript

var shadowScroll = document.querySelectorAll('.shadowScroll');
new ShadowScroll(shadowScroll, {
  prefix: 'shadowScroll'
});

Remember that instantiation needs to be done within a load or domready.

Options

Name Type Default Description
prefix string shadowScroll Class prefix for generated elements
noStyles boolean false Don't use the default CSS (generated by JS)

Requirements

ShadowScroll is framework-agnostic. No need for jQuery. It's written in CoffeeScript, and compiled into JavaScript.
